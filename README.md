# My React App

This is a simple React application that displays a "Hello World" message along with a spinning globe image in the center of the screen.

## Project Structure

```
my-react-app
├── src
│   ├── App.tsx          # Main component that renders the Hello World message and Globe component
│   ├── index.tsx        # Entry point of the React application
│   ├── components
│   │   └── Globe.tsx    # Component that displays a spinning globe image
│   └── types
│       └── index.ts      # TypeScript types and interfaces
├── public
│   └── index.html        # Main HTML file for the application
├── package.json          # npm configuration file
├── tsconfig.json         # TypeScript configuration file
└── README.md             # Documentation for the project
```

## Getting Started

To get started with this project, follow these steps:

1. Clone the repository:
   ```
   git clone <repository-url>
   ```

2. Navigate to the project directory:
   ```
   cd my-react-app
   ```

3. Install the dependencies:
   ```
   npm install
   ```

4. Start the development server:
   ```
   npm start
   ```

## Features

- Displays a "Hello World" message.
- Includes a spinning globe animation.
